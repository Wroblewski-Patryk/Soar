# Audit History

Audits, baselines, inventories, parity matrices, scan summaries, review
reports, and drift reports live here.

Use this folder when you need evidence of a past inspection. Current reusable
audit process documentation may still live in `docs/analysis/` when it defines
how to run future audits.

Typical files:
- `*-audit-*.md`
- `*-baseline-*.md`
- `*-inventory-*.md`
- `*-parity-*.md`
- scan and ledger reports
