# Release History

Release packets, final preflights, release gates, RC gate records, sign-off
history, activation packets, and release scorecards live here.

Use this folder when reconstructing a release decision or production readiness
claim. Current release runbooks and operator procedures belong in
`docs/operations/`.

Typical files:
- final preflights
- release gate outputs
- RC gate and sign-off records
- operator unblock packets
- release scorecards and activation packs
