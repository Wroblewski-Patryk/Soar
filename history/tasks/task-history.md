# Task History

Completed task contracts and execution packets live here.

Use this folder when you need to understand what an agent actually did during a
bounded mission. Do not treat a task file as current product or architecture
truth unless that truth was also normalized into `docs/` or `.agents/state/`.

Typical files:
- `*-task.md`
- bounded mission packets
- completed implementation or verification task records
