# Language Policy

## Repository Language
All repository artifacts must be written in English:
- documentation
- plans and checklists
- prompt templates
- architecture and decision records

## Communication Language
AI should communicate with the end user in the user's language.

## Mixed-Team Rule
If multiple collaborators use different languages:
- keep source files in English,
- keep user-facing conversation in the requester's language,
- summarize decisions in English in repository files.

## Enforcement
Before marking a task done, verify:
- new/edited docs are in English,
- project communication behavior follows user-language routing.
