# Planning

This folder contains active or durable planning sources only. Completed task
contracts live in [Task History](../../history/tasks/task-history.md), and old plans
or remediation notes live in [Plan History](../../history/plans/plan-history.md).

## Rule

New `*-task.md` files should be created in `history/tasks/`. Old plans,
closure notes, and remediation notes belong in `history/plans/`, then link
from active state files only while they remain relevant.
